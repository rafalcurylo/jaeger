1. build server application and create a docker image
Check docker-compose.yml for proper image version (of jaxserver4jaeger:X.Y)

cd jaeger/sb-rest-service
mvn clean install
docker image build -t sbrestservice4jaeger:1.1 .

2. build client application and create a docker image
cd jaeger/sb-rest-client
mvn clean install
docker image build -t sbrestclient4jaeger:1.1 .

3. run
cd jaeger/sb-rest-client
docker-compose up

4.test
Service:
http://localhost:8080/greeting?name=Test 

Client:
http://localhost:8087/callgreeting
